# X Anti-Spam Safety Rules

## Replies on X — Avoiding Spam Flags
Replies are strictly monitored. Adhere to the following guidelines when automating replies.

### Safe Reply Volume
| Account Type | Safe Daily Replies |
|---|---|
| New account (<30 days) | 5–10 |
| Personal/individual | 10–30 |
| Active creator/brand | 30–50 |
| Power user | 50–100 (with natural variation) |

### What Gets You Flagged Fast
**Content-based triggers:**
- Same reply text copy-pasted to multiple posts
- Replying with only a link (no context)
- Promotional replies to unrelated conversations
- Using the same hashtags in every reply

**Behavior-based triggers:**
- Replying to dozens of posts in a short burst (e.g., 20 replies in 20 minutes)
- Only replying to high-follower accounts to piggyback visibility
- Never getting replies *back* (one-way conversation pattern)
- Replying to strangers constantly with no mutual engagement history

### Best Practices
- **Space replies** at least 5–10 minutes apart during heavy activity (e.g., by limiting posts per script run via cron).
- **Vary your wording** — even if making the same point.
- **Mix reply types** — some short, some longer, some questions.
- **Engage back** — when people reply to your replies, respond.
- **Build context** — reply to accounts you already follow/interact with.

### The Core Rule
X looks for **reciprocal, varied, human-paced conversation**. Broadcast-like behavior will result in a flag regardless of volume.
